# LegisCidade

Portal de consulta à legislação municipal brasileira — site estático inspirado no leismunicipais.com.br, com paleta verde-esmeralda e logo personalizada.

## Recursos
- Busca por **Estado → Município** (26 estados + DF) e por **palavra-chave**
- Seções de recursos, municípios em destaque, estatísticas, planos e contato
- Layout responsivo (desktop, tablet e mobile)

## Tecnologia
HTML, CSS e JavaScript puros — sem dependências ou build.

## Como rodar localmente
Basta abrir o arquivo `index.html` no navegador, ou servir a pasta:

```bash
npx serve .
```

## Estrutura
- `index.html` — estrutura da página
- `styles.css` — estilos e paleta de cores
- `script.js` — busca interativa e formulários
- `assets/logo.svg` — logo personalizada
